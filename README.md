# TechSaber
Plataforma digital de aprendizado em tecnologia, desenvolvida como trabalho acadêmico (PIM III) no curso de **Análise e Desenvolvimento de Sistemas - 2026**.

---

## Sobre

O TechSaber+ é uma plataforma web educacional voltada ao ensino de tecnologia. Os usuários podem se cadastrar, acessar cursos organizados em módulos e aulas, realizar exercícios e acompanhar seu progresso de aprendizagem.

---

## Funcionalidades

- Cadastro e login com controle de acesso por perfil (Aluno, Professor, Administrador)
- Navegação por módulos e aulas sequenciais
- Marcação de aulas concluídas e acompanhamento de progresso
- Correção automática de exercícios objetivos
- Layout responsivo (desktop e mobile)

---

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/techsaber-plus.git

# Acesse a pasta
cd techsaber-plus

# Abra o arquivo principal no navegador
open index.html
```

> Nenhuma dependência externa necessária — o projeto roda com HTML/CSS/JS puro.
